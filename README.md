# Poe2OpenAI

poe官方sdk转换openai接口规范, 您必须拥有 poe 订阅会员权限, 然后到 [https://poe.com/api_key](https://poe.com/api_key) 页面拿到您的key   

注意, 如果您的会员到期, 此key会立即失效, 当您重新续费后, 此key会自动变更.   

无法在国内网络访问 poe ,因此请配置代理服务.

## 功能

- [x] 流式输出
- [x] 非流式输出 (仅用于测试连通性)
- [x] http代理
- [ ] 模型列表
- [ ] Function Call
- [ ] 图片解析
- [ ] Embedding
